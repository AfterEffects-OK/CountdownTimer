# **⏳ 動画再生用 カウントダウンタイマー**

このアプリケーションは、動画編集やライブ配信などで使用することを想定した、シンプルかつ直感的な操作が可能なカウントダウンタイマーです。

## **✨ 特徴**

* マウスホイールやスマートフォンのフリック操作（上下スワイプ）による簡単な時間設定  
* 設定時間から特定の時間だけ遅延させて開始できるオフセット（遅延時間）設定機能  
* 残り時間に応じた視覚的な色の変化による通知

## **🚀 1\. 基本的な使い方**

### **時間設定**

1. 画面下部の「タイマー設定 (時:分:秒)」エリアで、カウントダウンしたい時間を設定します。  
2. 以下の「特徴的な操作」セクションを参照して、入力フィールドを操作できます。

### **遅延時間設定 (オプション)**

* 「再生開始遅延時間 (動画再生済み時間)」は、カウントダウンの開始時間を設定した合計時間から差し引くオフセット（遅延）時間を設定します。  
* **例:** 「タイマー設定」が 01:00 (60秒)、「遅延時間」が 00:10 (10秒) の場合、タイマーは **50秒** からカウントダウンを開始します。  
* すでに一部が再生されている動画の経過時間に合わせる際に便利です。

### **タイマーの操作**

| ボタン | 状態 | 動作 |
| :---- | :---- | :---- |
| スタート (▶️) | 初期/停止時 | 設定時間からカウントダウンを開始します。 |
| 一時停止 (⏸️) | 実行中 | タイマーを一時的に停止します。 |
| 再開 (▶️) | 一時停止中 | 停止した時点からカウントダウンを再開します。 |
| リセット (🔄) | 実行中/停止中 | タイマーを停止し、初期設定時間に戻します。 |

## **🕹️ 2\. 特徴的な操作 (ホイール・フリックによる連動操作)**

入力フィールドは、PCとモバイルの両方で直感的に操作できるように設計されています。

| 操作方法 | 動作 | 効果 |
| :---- | :---- | :---- |
| マウスホイール (PC) | 入力フィールド上でホイールを上下に回す。 | 値が ±1 されます。分と秒は、0や59になったときに隣の単位と連動して繰り上がり/繰り下がりします。 |
| 上下フリック (スマホ) | 入力フィールド上で指を上下にスワイプ（フリック）する。 | マウスホイールと同様に、値が ±1 され、連動して繰り上がり/繰り下がりします。 |

## **🎨 3\. カウントダウン表示の色のルール**

残り時間に応じて、タイマーの数字の色と背景色が変わることで、視覚的に終了を通知します。

| 残り時間 | 数字の色 | 背景色 | 備考 |
| :---- | :---- | :---- | :---- |
| 11秒以上 | 白 (text-white) | 濃い灰色 (bg-gray-900) | 通常の表示状態です。 |
| 10秒〜1秒 | 黄色 (text-yellow-400) | 濃い灰色 (bg-gray-900) | 終了が近いことを示します。点滅は**しません**。 |
| 0秒 | 赤（点滅） (text-red-500) | 暗い灰色 (bg-gray-800) | カウントダウン終了を示します。赤色の点滅アニメーションが実行されます。 |

**📝 注意事項**

* タイマー動作中は、時間設定および遅延時間の設定フィールドはロックされ、変更できません。  
* 設定した遅延時間はブラウザのローカルストレージに保存され、次回アクセス時にも同じ値が使用されます。  
* アラートメッセージ（画面右上に表示）は、操作結果やバリデーションエラーを通知するために使用されます。
